# RGSS Simple Scenario Scripter (RGSS S3)

#### Example:

```yaml
はじまり:
- a1: ここはどこだ…
- wait: 10
- 答えを知りたいか……？
- yes:
  - 教えてくれ…！
  no:
  - 特に興味はないな…
```

## Commands

* `wait`: Wait for frame count.
* `yes`/`no`: Ask Yes or No prompt.
* `a<num>`: Show actor image.
* `fadein`/`fadeout`: Screen fade in/out
* `get_weapon`/`get_item`/`get_wear`..
